# DJI Footer 詳細結構分析

根據截圖，DJI Footer 底部的精確排版如下：

## 最底部區域（Copyright 區域）

### 左側區域：
1. **第一行**：Logo 圖示（紅色 DJI Logo）
2. **第二行**：連結群組
   - 關於我們
   - 聯絡我們  
   - 招聘精英
   - 成為合作夥伴
   - RoboMaster
3. **第三行**：法律連結
   - 私隱權政策
   - 使用條款
4. **第四行（最底部）**：
   - **Copyright © 2025 DJI 大疆創新 版權所有**
   - 網站問題反饋

### 右側區域：
1. 社群媒體圖示（Facebook、Twitter、Youtube、Instagram）
2. 語言/地區選擇器：🌐 中國台灣 / 繁體中文

## 關鍵發現

**Copyright 的精確位置：**
- 位於 Footer **最底部**的**最左側**
- **獨立一行**（不與私隱權政策、使用條款在同一行）
- 在私隱權政策/使用條款的**下方**
- 靠左對齊
- 與「網站問題反饋」在同一行

## 完整排版順序（由上到下）：

```
左側：
├─ Logo
├─ 關於我們 | 聯絡我們 | 招聘精英 | 成為合作夥伴 | RoboMaster
├─ 私隱權政策 | 使用條款
└─ Copyright © 2025 DJI 大疆創新 版權所有 | 網站問題反饋

右側：
├─ [社群媒體圖示]
└─ 🌐 中國台灣 / 繁體中文
```

## 與目前 Apolnus 的差異

目前 Apolnus 將 Copyright 與「私隱權政策」和「使用條款」放在同一行。

**應該改為：**
- 私隱權政策和使用條款在一行
- Copyright 在下方獨立一行
- 語言選擇器在右側
